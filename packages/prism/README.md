# @chiffon/prism

Fork of [Prism](https://prismjs.com/)

- Migrates to ES6
- Uses import statements instead of own dependency mechanism
- Removes autohighlight behavior
- Removes Prism from `global`

## Installation

npm:

```sh
npm install @chiffon/prism
```

yarn:

```sh
yarn add @chiffon/prism
```
